# IT1214-Student-Grade-System
## 📌 Project Purpose
This is a simple Java console-based application developed for managing student records in an educational institute.  
It allows teachers to store student information, search records, and calculate average marks using Object-Oriented Programming (OOP) concepts.

## 🚀 Features
- Add student details (ID, Name, Marks)
- Display all student records
- Search student by ID
- Calculate average marks
